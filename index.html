Lightbringer

A setup analyzer for A Game of Thrones: The Card Game - 2nd Edition

How it works

Lightbringer analyzes the setups for a A Game of Thrones: The Card Game - 2nd Edition deck. Paste a ThronesDB deck URL and click Analyze to test the setup of 5000 opening hands.

For each hand, every valid setup is evaluated. The best setup is selected according to configurable ranking criteria — see Setup Ranking Criteria below. If the best setup is considered poor (failing any configured requirement), a mulligan is taken — a fresh hand is drawn and evaluated once more.

The results include summary statistics, several distribution charts, and a simulation log of up to 100 hands. Use the Configuration section below to define what counts as a poor setup, to mark economy, key, avoidable, and restricted cards, and to re-order the ranking criteria.
